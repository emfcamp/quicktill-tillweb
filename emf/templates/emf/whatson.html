What's powering this?

The tills are run using free software.

  • The till software
  • This web service
  • The display boards boot over the network and fetch https://bar.emfcamp.org/display/

Pull requests welcome!

A simple JSON interface is available:

  • https://bar.emfcamp.org{% url "locations-json" %} lists the groups of stock that are on sale at the moment
  • https://bar.emfcamp.org/location/[name].json lists stock on sale in a location, for example the Bar location
  • https://bar.emfcamp.org{% url "stock-json" %} lists everything we have left in stock.
  • https://bar.emfcamp.org{% url "progress-json" %} compares how much we've actually drunk with what we predicted before the event...
